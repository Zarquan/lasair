Lasair is a broker for astronomers studying transients and variables. It is being developed as part of the LSST:UK Phase A programme through a collaboration between the University of Edinburgh and Queen's University, Belfast with funding from grants ST/N002512/1 and ST/N002520/1, respectively. To help prototype the functionality needed to process alerts from LSST, Lasair is currently ingesting and presenting alerts from the public stream of the ZTF survey. Its capabilities will develop incrementally, guided by user feedback, so please try it and tell us what more you need to do your science.

You can query the alerts by selecting "Query ZTF" at the left, you can see the coverage of the ZTF survey each night, and you can -- with a login -- query the database through a Jupyter notebook interface.

Lasair started ingesting ZTF on 28 May 2018, and now has {{ n_candidates }} candidate alerts.

Some functions of this website require a login. If you want to use these functions, use the "signup" link at the top left of this page.

Questions and enquiries, please click "Contact". Make suggestions for improvements and fixes at the Lasair github issues page.
